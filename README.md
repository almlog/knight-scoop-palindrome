# 音素回文発見プロジェクト

**探偵ナイトスクープで紹介された「オオエンマハンミョウ」に続く音素回文を、日本語31万語超から探索するプロジェクト**

🔗 **デモ**: [knight-scoop-palindrome.streamlit.app](https://knight-scoop-palindrome.streamlit.app/)

---

## 音素回文とは？

音素（音の最小単位）に分解したとき、前から読んでも後ろから読んでも同じ配列になる言葉。

```
オオエンマハンミョウ → o o e m m a h a m m y o o
                       ← o o y m m a h a m m e o o  ← 同じ！
```

探偵ナイトスクープの放送で「オオエンマハンミョウ」が逆再生しても同じに聞こえることが紹介され、不思議な気持ちともっと見つけてみたいという好奇心が芽生えました。本プロジェクトは「他にも同じ性質を持つ日本語があるのでは？」という疑問から出発し、それをAIとプログラミングで解決していくプロジェクトです。

## 調査結果

| 項目 | 数値 |
|------|------|
| 総調査ワード数 | **314,940** |
| 音素回文として発見 | **280 件**（厳密検証済み） |
| ヒット率 | 約 0.09% |

## 主な機能

- **自由入力**: 好きな言葉を入力して通常再生・逆再生を聴き比べ
- **音声検証**: gTTSによる音声合成 + pydubによる逆再生で、音素回文を耳で確認
- **発見ワード一覧**: 品詞カテゴリ別に回文ワードを表示、再生ボタン付き
- **全データ閲覧**: 31万語超の調査データをカテゴリ別フィルター・検索可能
- **音素ビジュアライザ**: 音素の順方向・逆方向を視覚的に比較

## 技術スタック

| 技術 | 用途 |
|------|------|
| [Streamlit](https://streamlit.io/) | Web UI |
| [gTTS](https://gtts.readthedocs.io/) | 音声合成（Google Text-to-Speech） |
| [pydub](https://pydub.com/) | 音声の逆再生処理 |
| [pandas](https://pandas.pydata.org/) | データ処理 |
| ffmpeg | 音声フォーマット変換 |

## ローカル実行

```bash
pip install streamlit pandas gTTS pydub
streamlit run app.py
```

※ ffmpeg のインストールが別途必要です。

## データ

- `knight_scoop_all_keywords_investigation.csv` — 主要調査データ（31万語超）

## このプロジェクトについて

毎週の放送を家族そろって観るのが我が家の楽しみでした。いつか何かの形で、この大好きな番組に関われたら──そんな夢をずっと抱いていました。

「オオエンマハンミョウ」の回を観たとき、心の底から感動し、**「これならば自分のプログラミングの経験が役に立てる」** と確信。AIとプログラミングを駆使して、日本語314,940語の音素解析に挑みました。

コーナーの最後、田村探偵が「もし暇な方がいたら〜」とおっしゃっていました。私は現在育児休暇中。絶賛暇。ということもありチャレンジしました。
